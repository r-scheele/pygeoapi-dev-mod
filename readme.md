
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/r-scheele/pygeoapi-dev-mod.git
   ```
2. Install PIP packages
   ```sh
   poetry install
   ```
3. Set environment variables in`example-config.yml`
   ```sh
   export PYGEOAPI_CONFIG=example-config.yml
   ```

3. run the server
   ```sh
   python3 run.py
   ```
